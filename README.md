# Instrucciones de compilación
Este proyecto está hecho en Java, por lo que necesitas tener instalado un JDK.

Para compilar el proyecto desde la raíz del repositorio, ejecuta:

```bash
javac src/main/java/*.java
```

Para ejecutar el programa después de compilar, usa:

```bash
java -cp . src.main.java.Main
```

Si quieres volver a compilar después de hacer cambios, repite primero el comando de `javac` y luego el de `java`.
